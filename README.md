# Shop App

### Çalıştırma Gereksinimleri
* .net 3.1.101 ve üzeri
* node.js


### İlk Kurulum
```bash
    npm install
```

### Çalıştırma(webapi ve webui)
```bash
    dotnet run
```

### Kullanılan Teknolojiler
* MVC
* Static Files
    * wwroot
    * node_modules
* Newtonsoft.Json
* Validation
    * Model
    * Client
    * Business
* File Upload
* HTML Editor
* Asp.Net Core Identity
* Csrf Token
* Email Sender(smtp)
* Datatable js
* Iyzico Sandbox API
* UnitOfWork Pattern
